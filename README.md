### 这是 MINDPIN KC 知识平台的演示中心工程，该工程用于以下目的：

- 支持产品特性的在线演示
- 用来实验一些具有探索性质的设计思路
- 用来实验一些最新的开发特性


### 运行环境

- ruby 2.2.5


### 运行方法

```
bundle install
npm install
rails s -b 0
```
